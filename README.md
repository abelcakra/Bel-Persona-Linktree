# Bel Persona Finder & Linktree

Website interaktif berbasis HTML, CSS, dan JavaScript yang menggabungkan:
- 🎭 Persona Generator (Anime Theme)
- 👤 Linktree Profile
- 🔊 Backsound interaktif
- 📄 Export hasil ke PDF

---

## 🚀 Fitur Utama

### Persona Finder
- Input: Nama, Hobi, Angka
- Output: Persona anime unik
- Logic berbasis kondisi (if-else)

### ✨ Special Feature
- Jika nama = **Abel**
  - 🐒 Persona berubah menjadi: **The Monkey King (Sun Wu Bel)**
  - 🔊 Backsound otomatis dimainkan
  - 🌌 Efek visual muncul

### 👤 Profile Linktree
- Menampilkan:
  - Foto profile
  - Nama (dinamis)
  - Link sosial media

### 📄 Export PDF
- Hasil persona bisa di-download
- Desain PDF sudah rapi & modern

---

## 🛠️ Teknologi yang Digunakan

- HTML5
- CSS3 (Flexbox & Responsive)
- JavaScript (DOM Manipulation)
- jsPDF (Export PDF)

## ▶️ Cara Menjalankan

1. Download / Clone repository
2. Pastikan semua file ada dalam satu folder
3. Buka `index.html` di browser

---

## 🎮 Cara Menggunakan

1. Masukkan:
   - Nama
   - Hobi
   - Angka (1–10)
2. Klik **"Temukan Persona"**
3. Hasil akan muncul
4. Klik **"Download PDF"** untuk menyimpan
